#### Wiring

|GPS Neo 6M Module|Arduino Uno R#|
|--|--|
|VCC|5V|
|GND|GND|
|RX Pin|RX<-0|
|TX Pin|TX<-1|

|GSM sim900A module|Arduino Uno R3|
|--|--|
|VCC|5V|
|GND|GND|
|RX Pin|RX|
|TX Pin|TX|

#### Note
`Before Upload, Unplug the RX TX Pin GPS Neo 6M Module`
